## About

Appointment lookup system designed to replace manual phone-based verification with a self-service digital interface.

Supports 24/7 patient access to appointment checking and basic healthcare information.

<br>

## Features

- 24/7 appointment self-check  
- Telemedicine information page  
- Service information page  
- Official contact information  
- FAQ section  

<br>

## Tech Stack

- Next.js  
- Google Sheets  

<br>

## Impact

- Reduced dependency on phone-based appointment verification  
- Improved patient accessibility for 24/7 service information  
- Simplified appointment inquiry workflow
